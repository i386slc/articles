–ü–æ—á–µ–º—É, –æ—Ç–≤–µ—Ç –ø—Ä–æ—Å—Ç, –ø–æ—Ç–æ–º—É —á—Ç–æ –º–æ–∂–µ—Ç –Ω–µ –±—ã—Ç—å –Ω–∏–∫–∞–∫–æ–≥–æ –ø–æ–∏—Å–∫–∞, –∫–æ—Ç–æ—Ä—ã–π –º–æ–≥ –±—ã –ø–æ–ª—É—á–∏—Ç—å –¥–∞–Ω–Ω—ã–µ –≤ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏–∏ —Å –≤–∞—à–∏–º–∏ —Å–ª–æ–∂–Ω—ã–º–∏ —Ç—Ä–µ–±–æ–≤–∞–Ω–∏—è–º–∏.

–î–∞–≤–∞–π—Ç–µ –ø—Ä–æ–≤–µ—Ä–∏–º –æ—á–µ–Ω—å –ø—Ä–æ—Å—Ç–æ–π –ø—Ä–∏–º–µ—Ä –∏–∑ Djangodocs, –∫–æ—Ç–æ—Ä—ã–π –¥–∞—Å—Ç –≤–∞–º –¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏, —á—Ç–æ–±—ã –Ω–∞—á–∞—Ç—å —Ä–∞–±–æ—Ç—É –Ω–∞–¥ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–º–∏ –ø–æ–∏—Å–∫–∞–º–∏.

–ú—ã –Ω–∞–ø–∏—à–µ–º —Å–æ–±—Å—Ç–≤–µ–Ω–Ω—ã–π –ø–æ–∏—Å–∫, –∫–æ—Ç–æ—Ä—ã–π –±—É–¥–µ—Ç —Ä–∞–±–æ—Ç–∞—Ç—å –ø—Ä–æ—Ç–∏–≤–æ–ø–æ–ª–æ–∂–Ω–æ **exact**. `Author.objects.filter(name__ne='Jack')` –ø–µ—Ä–µ–≤–µ–¥–µ—Ç –≤ SQL:

```python
from django.db.models import Lookup

class NotEqual(Lookup):
    lookup_name = 'ne'
    
    def as_sql(self, compiler, connection):

        rhs, rhs_params = self.process_rhs(compiler, connection)
        params = lhs_params + rhs_params
        return '%s <> %s' % (lhs, rhs), params
```
